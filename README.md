<h1>ハッピーエディター</h1>
srt形式の字幕を生成します。<br>
動画の再生に合わせてボタンを押下することで楽にタイムスタンプを記述することができます。
<img style="width:400px" src="https://github.com/takashi-koshiba/happy-editor/blob/master/%E3%82%AD%E3%83%A3%E3%83%97%E3%83%81%E3%83%A3.PNG">
<h2>使用方法</h2>
アプリ内のヘルプをご覧ください。

<h2>準備</h2>
vlcのインストールが必要です。<br>
必ずwin32版をインストールしてください。<br>
インストール先はデフォルトの「C:\Program Files (x86)\VideoLAN」にしてください。

<h2>動作確認ができた環境</h2>
OS:windows10<br>
vlcバージョン:vlc-3.0.17.4-win32
